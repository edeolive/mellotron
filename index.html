Code: mellotron

Overview

For project 3, I wrote a python program that estimates the melody from a sound file and uses that to output a track using sounds from a Mellotron.s I chose to emulate Mellotron sounds with this software because of the Mellotron's origins as one of the first samplers. Unlike a modern sampler which plays back digital audio samples, the Mellotron played analog audio tapes. Essentially, this program is a software sampler that uses samples from an analog sampler.
At first I used a peak finding algorithm to approximate melodies, but found better performance using Spotify's Basic Pitch library. In my testing Basic Pitch still has some issues with guessing guitar melody, especially when the signal contains other effects like distortion. It is also very sensitive to noise and, so my sloppy playing really sticks out LOL. Performance is likely better with something like a piano but I don't have one to test.

The program depends on the following python libraries: basic-pitch, pretty_midi, wave, soundfile. I didn't have time to write a nice CLI for this program, so usage is simply modifying and running the script:
python main.py
